# 🦅 Анализ стратегий в эволюционной игре "Ястребы и Голуби"

> Стохастическая симуляция эволюционной динамики популяции на основе теории игр  
> Курсовая работа 

Симуляция показывает, как частотно-зависимый отбор и случайные взаимодействия приводят к установлению **эволюционно стабильного соотношения** стратегий, даже без центрального управления.

## ✨ Возможности
- Поколенная эволюция с взвешенным случайным отбором
- Динамический график изменения численности стратегий
- Сравнение с теоретическим равновесием (ESS)
- ️Гибкая настройка параметров симуляции через константы в начале файла
- Вывод статистики в консоль в реальном времени

##  Быстрый старт

### Требования
- Python 3.8+
- Библиотека `matplotlib`

### Установка и запуск
```bash
# 1. Клонируй репозиторий (или скачай ZIP)
git clone https://github.com/ТВОЙ_НИК/Evo_Analysis_kursach.git
cd Evo_Analysis_kursach

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Запусти симуляцию
python Evo_Analysis_kursach.py
